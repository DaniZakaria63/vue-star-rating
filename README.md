# Vue-star-rating
A simple, customisable star rating component for vue apps.

  - Counter
  - Descriptions (soon)
  
![](https://s4.gifyu.com/images/rating.gif)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
